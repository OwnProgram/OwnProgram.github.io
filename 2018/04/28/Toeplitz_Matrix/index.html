Toeplitz Matrix

A matrix is Toeplitz if every diagonal from top-left to bottom-right has the same element.
Now given an M x N matrix, return True if and only if the matrix is Toeplitz.

solution 1

The idea of solving the problem

依次扫描矩阵中的所有元素，并把当前的元素s[i][j]与其右下角元素s[i+1][j+1]比较是否相同。若相等，则s的值为“true”,若在扫描过程中一旦遇到不相等的情况，则将s的值设为“false”，并结束该程序。

solution 2

The idea of solving the problem

  1. 把矩阵中每条对角线上的第一个元素设为该对角线的默认值，将其加入字典。
  2. 同一条对角线上的元素,其 行编号减列编号 的值都相等。
